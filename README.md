# Extraordinary Energy : Creazy

## 概要
[Extraordinary Energy : Modern](https://github.com/sponeru/1000x-mekanism-1.20)のエッセンスを引き継ぎ、1.21.1にポートしたModpackです。

## 開発者の方へ

インストールしたMinecraftディレクトリの下へ

```bat
mklink /d config <クローン先のフォルダー>/config
mklink /d kubejs <クローン先のフォルダー>/kubejs
mklink /d defaultconfigs <クローン先のフォルダー>/defaultconfigs
```

を実行し、連携させてください
